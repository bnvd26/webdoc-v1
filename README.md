
# webdoc-v1

webdoc using symfony &amp; vueJS

# With homebrew

### Install composer

  

`brew install composer`

  
  

### Install yarn

  

`brew install yarn`

  
  

### webpack

  

`composer require symfony/webpack-encore-bundle`

  
  

### doctrine

  

`composer require symfony/orm-pack`

  
  

### init database

  

`php bin/console doctrine:database:create`
